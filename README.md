## Overview

**SmokeLog Discord Bot** allows users to view personal usage statistics, leaderboards, and manage
reminders directly from Discord, with all data securely backed by SmokeLogâ€™s
backend infrastructure.

## Features
- Personal usage statistics (concentrates, flower, carts)
- Public leaderboards with category-based rankings
- Persistent reminders with timezone support
- Quote saving via message context menu
- Deleted message snipe utility
